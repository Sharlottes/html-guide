속성

 <tag pro="para">inside</tag>:
  • tag: 태그 이름입니다. </tag>로 태그를 닫을 수 있지만, 일부 태그는 생략할 수 있습니다.
  • pro: 속성 이름입니다.
  • "para": 속성값입니다. 큰따음표는 필수가 아니지만, 속성의 구분은 띄워쓰기로 구분되므로 큰따음표로 속성값에 의해 띄워쓰여지는 문제를 막을 수 있습니다.
  • inside: 내용입니다.


문서 메타데이터

 <!DOCTYPE html>: 현재 문서가 HTML5 문서임을 명시합니다.
<head>: HTML 문서의 메타데이터를 설정합니다.
<title>: HTML 문서의 제목을 정의하며, 브라우저의 툴바나 즐겨찾기, 검색엔진 결과의 제목에 표시됩니다.
<meta>:
  • charset 속성으로 인코더(ex: ascii, ansi, iso-8859-1, utf-8)를 설정할 수 있습니다.
  • name와 description 속성으로 이름과 설명을 넣어서 검색 엔진에 더 잘 노출되게 할 수 있습니다.
  • lang 속성으로 페이지의 언어를 설정할 수 있는데, 이는 <span> 태그에도 쓰여 각 언어에 따른 문자를 설정할 수 있습니다.
  • 그 외에도 여러 SNS(ex: facebook, twitch)에서 독점 속성을 가지고 있습니다.

목록

 <ul> =Unordered List
  • 이 태그는 순서가 없는 목록을 표시합니다.
  • 상위 목록은 <ul>, 하위 목록은 <li> 태그를 사용합니다
  • type 속성을 통해서 모양을 바꿀 수 있으며 disc, circle, square 총 3개가 있습니다
  • 이 모양이 disc 입니다
  • 이 모양이 circle 입니다
  • 이 모양이 square 입니다
    • 하위 목록에 <ul> 태그를 또 사용하여 이처럼 표현할 수 있습니다.
    • <ul> 와 <li> 태그 둘 다 type 속성이 있습니다.
<ol> =Ordered List
  1. 이 태그는 순서가 있는 목록을 표시합니다.
  2. 상위 목록은 <ol>, 하위 목록은 <li> 태그를 사용합니다
  3. type 속성을 통해서 순서 문자를 바꿀 수 있으며 1, A, a, i, I 총 5개가 있습니다
  4. 이 문자가 1 입니다
  5. 이 문자가 A 입니다
  6. 이 문자가 a 입니다
  7. 이 문자가 i 입니다
  8. 이 문자가 I 입니다
  9. <li> 태그는 value, <ol> 태그는 start 속성을 통해 특정 순서로 이동할 수도 있습니다
  10. 시작 지점을 설정하는 것이므로 유의해야 합니다.
<dl> =Difinition List
이 태그는 용어의 정의를 나열할 때 쓰입니다
<dt> 태그로 용어를, <dd> 태그로 용어의 속뜻을 적습니다.
type 같은 속성은 따로 없습니다.


이미지

 <img> 태그는 src 속성에 이미지 경로를 넣어서 이미지를 불러올 수 있습니다
경로를 표시하는 방법엔
  1. 현재폴더: .
  2. 상위폴더: ..
  3. 최상위폴더: /
  4. 하위폴더: 폴더명
ex) ./htmllogo.jpg - 상위 폴더
/htmllogo.jpg - 최상위 폴더

스타일

 스타일은 주로 CSS를 통해서 꾸며진다. CSS는 스타일 표 언어이며, <link>를 통해 스타일을 미리 정의하여 가져올 수 있거나 <style>를 통해 일회용으로 바로 만들 수도 있다.

<style>는 거의 대부분의 태그의 속성이며, 선택자와 중괄호를 제외한 부분이 속성값이다. (style: "color:red;")

<link>는 CSS파일을 통째로 불러들이는 방식이며, <link href="파일명" rel="stylesheet" type="text/css">를 <head> </head> 사이에 넣으면 된다.
CSS 파일에선
선택자 {
속성: 속성값;
};
와 같이 작성합니다.
이때 지켜야 할 규칙이 있습니다.
  • 중괄호는 필수입니다.
  • 각 속성을 구분하기 위해서 :와 ;는 필수입니다.
선택자는 이 속성들스타일을 적용할 대상을 말하며, HTML 태그, id&class 이름 등이 해당됩니다.
콤마를 통해 선택자를 2개 이상 설정할 수도 있습니다.

선택자 이름 선택하는 것 예시
요소 선택자 (때때로 태그 또는 타입 선택자라 불림) 특정 타입의 모든 HTML 요소. p
<p>를 선택
아이디 선택자 특정 아이디를 가진 페이지의 요소
(주어진 HTML 페이지에서, 아이디당 딱 하나의 요소만 허용됩니다).		 #my-id
<p id="my-id"> 또는 <a id="my-id"> 를 선택
클래스 선택자 특정 클래스를 가진 페이지의 요소
(한 페이지에 클래스가 여러번 나타날 수 있습니다).		 .my-class
<p class="my-class"> 와 <a class="my-class"> 를 선택
속성 선택자 특정 속성을 갖는 페이지의 요소. img[src]
<img src="myimage.png"> 를 선택하지만 <img>는 선택 안함
수도(Pseudo) 클래스 선택자 특정 요소이지만 특정 상태에 있을 때만, 예를 들면, hover over 상태일 때. a:hover
<a> 를 선택하지만, 마우스 포인터가 링크위에 있을 때만 선택함

CSS 색상값

일부 속성들(ex: color, background-color)의 속성값은 색상값이며, 별도의 키워드를 지원합니다.
  • 16진수 6자리: #rrggbb
  • 16진수 3자리: #rgb
  • 10진수; rgb(255, 0, 0)
  • 퍼센트: rgb(100%, 0%, 0%)
  • 그 외 고유 키워드들


CSS 길이단위

일부 속성들(ex: width, height)의 속성값은 길이값이며, 별도의 단위를 지원합니다.
  • 절대 단위
    • mm(밀리미터)
    • cm(센티미터)
    • in(인치, =2.54cm)
    • pt(포인트, =1/72in)
    • pc(파이카, 12pt)
  • 상대 단위
    • px(픽셀)
    • em(부모요소의 폰트크기 속성값을 1로하는 단위)
    • ex(부모요소의 소문자 'x'의 높이를 1로하는 단위)


CSS URL

일부 속성들(ex: background-color)의 속성값은 이미지같은 리소스이며, url(경로)함수를 통해 리소스를 불러들일 수 있습니다.

참고: 자식요소는 부모요소의 스타일을 함께 상속받습니다.
